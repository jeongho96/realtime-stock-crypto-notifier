실시간 주식/코인 시세 알림 서비스

이 프로젝트는 외부 시세 API의 데이터를 주기적으로 수집하고, 사용자가 설정한 조건에 따라 WebSocket을 통해 실시간 알림을 전송하는 백엔드 시스템입니다. 백엔드 개발자에게 필요한 실시간 데이터 처리, 외부 API 연동, 비동기 통신, 스케줄링 및 RESTful API 설계 역량을 종합적으로 보여줍니다.

주요 기능

📡

실시간 데이터 수집

외부 API를 주기적으로 호출하여 최신 시세 정보를 실시간으로 가져옵니다.

📝

사용자 알림 설정

REST API를 통해 사용자가 원하는 종목과 알림 조건을 자유롭게 설정합니다.

⚡️

실시간 알림 전송

설정된 조건 충족 시 WebSocket을 통해 클라이언트에게 즉시 알림을 보냅니다.

🕒

데이터 스케줄링

@Scheduled 어노테이션으로 정해진 간격마다 시세 확인 및 알림 검사를 자동 수행합니다.

API 탐색기

REST API: 알림 설정

POST /api/notifications
알림 설정 등록

Request Body:

{
    "market": "KRW-BTC",
    "targetValue": 50000000.0,
    "conditionType": "PRICE_REACHED",
    "operator": "GREATER_THAN"
}
GET /api/notifications
알림 목록 조회

Response Body:

[
    {
        "id": "...",
        "market": "KRW-BTC",
        ...
    }
]
DELETE /api/notifications/{id}
알림 설정 삭제

Path Parameter:

id: "알림 설정 ID"

WebSocket API: 실시간 알림

Endpoint:

/ws/notifications

구독 채널:

/user/queue/notifications

수신 메시지 예시:

{
    "type": "NOTIFICATION_ALERT",
    "market": "KRW-BTC",
    "currentPrice": 50100000.0,
    "alertMessage": "KRW-BTC 가격이 5천만원을 돌파했습니다!",
    "timestamp": "..."
}

향후 개선 방향

🔐 인증/인가 기능

JWT 기반 사용자 인증을 추가하여 개인화된 알림 서비스를 제공합니다.

📦 메시지 큐 도입

Kafka/RabbitMQ를 도입하여 시스템 확장성과 안정성을 높입니다.

🐳 Docker 컨테이너화

프로젝트를 Docker 컨테이너로 만들어 배포 용이성을 확보합니다.

☁️ 클라우드 배포

AWS, GCP 등 클라우드 환경에 배포하여 실제 서비스 운영 경험을 쌓습니다.

📊 모니터링 구축

Prometheus, Grafana로 시스템 상태 및 성능을 실시간 모니터링합니다.

🚀 성능 최적화

DB 인덱싱, 캐싱(Redis) 등을 적용하여 응답 속도와 처리량을 개선합니다.
